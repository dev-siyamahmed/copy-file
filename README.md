# North Star Journal

A one-page editorial blog site built with Next.js and Tailwind CSS.

## Folder Map

- `src/app/page.js` is the main page, similar to an `index.html` homepage.
- `src/app/layout.js` is the shared HTML shell.
- `src/components/` holds reusable UI blocks.
- `src/data/site.js` keeps the copy in one place so it is easy to edit.

## Run It

1. Install dependencies with `npm install`.
2. Start the site with `npm run dev`.
3. Open the local URL shown in the terminal.

## Edit It

- Change the text in `src/data/site.js` to replace the article copy.
- Reorder or duplicate cards in `src/app/page.js`.
- Add more sections inside `src/components/ContentSection.js` if you want a longer article.
